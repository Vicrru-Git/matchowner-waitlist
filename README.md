# matchowner-waitlist

Webapp/landing simple para la waitlist de Match Owner. Frontend-only, mockeado.

3 pantallas: `/` (landing/hero) → `/signup` (registro) → `/dashboard` (cola + pregunta diaria + invite).

Stack: Next.js 16 (App Router) + TypeScript + Tailwind v4 + Framer Motion + Lucide React.
Design tokens y fuentes reusan los de `../matchowner-inmobiliarias/`.

## Dev

```bash
npm install
npm run dev
```

App en `http://localhost:3000`.
